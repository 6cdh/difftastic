## Nodes

This page contains all visible nodes in yaml format.

```yml
- comment # all comments, or read-case-sensitive parameter
- extension # #lang or #reader
- dot # .

- boolean
- string
- escape_sequence # escape sequence in string
- here_string
- byte_string
- character
- number
- symbol

- keyword # for example, #:abc
- regex
- box
- graph
- structure
- hash # hash table
- quote

- list
- vector
```
